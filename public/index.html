Welcome

Hey there! This is a project I made in the MEAN stack (MongoDB, Express, Angular, and Node) that builds and displays a user generated gallery of .gifs and commentary.

Feel free to search through what's already here via the catalogue view.

Or, check out the stream view to cycle through individual .gifs that I'm pulling from Giphy's API. Click the arrow to go to another one. If you find one you'd like to say something about, add commentary and save it to the rest of the catalogue.

Thanks for checking it out!
